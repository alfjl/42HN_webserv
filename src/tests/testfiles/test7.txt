<p>Let's use my Perl socket test program to send HTTP 1.1 requests to some popular Web servers.
</p>

<p>HTTP 1.1 request requires the "Host" header line. I need to create a new request file 
with an extra line, http_1_1_get_perl.req. Don't forget to have 2 new line (\n) characters to end the file.
</p>
<pre>
GET / HTTP/1.1
Host: www.perl.org


</pre>

<p>Here is my test with www.perl.org. 
It used to be working in 2009. But now it is not responding. 
I have to break the call with &lt;Ctrl-C&gt;:
</p>
<pre>
C:\herong&gt;perl SocketRequestResponse.pl www.perl.org 80 \ 
   http_1_1_get_perl.req http_get.res

&lt;Ctrl-C&gt;
</pre>

<p>My second test is www.w3.org. 
I used to be working in 2009. But now it is returning 
the 400 error:
</p>
<pre>
C:\herong&gt;more http_1_1_get_w3.req
GET / HTTP/1.1
Host: www.w3.org

C:\herong&gt;SocketRequestResponse.pl www.w3.org 80 \
   http_1_1_get_w3.req http_get.res

C:\herong&gt;more http_get.res
HTTP/1.1 400 Bad Request
